# Code of Conduct

Heavy Coder follows the Contributor Covenant Code of Conduct, version 2.1.

## Our pledge

We pledge to make participation in this project harassment-free for everyone.

## Expected behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experience levels.
- Accept constructive feedback gracefully.
- Focus on what is best for the community and maintainers.

## Unacceptable behavior

- Harassment, insults, or demeaning comments.
- Publishing private information without permission.
- Abuse of project automation or credentials.

Report concerns through the repository security contact or maintainers.
